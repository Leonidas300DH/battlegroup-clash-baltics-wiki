# Russia Unit Catalog

All [[russia|Russian]] [[force-elements|Force Element (FE)]] and [[support-assets|Support Asset (SA)]] types. Russian counters are **red/salmon**. Russia generally has more units but lower [[morale|Troop Quality (TQ)]] and [[combat-effectiveness|Resilience]] than [[nato|NATO]].

---

## Platoon-Sized Force Elements ([[force-elements|Platoon (PL)]] — 3 dots)

### Infantry Platoon (Foot [[force-elements|Platoon (PL)]])

Russian infantry. Like NATO infantry, required for [[assault|Assaults]] and cannot target vehicles without [[atgm|Anti-Tank Guided Missiles (ATGMs)]] (RPG or Stabber).

| Stat | Full | Half Resilience |
|------|------|----------------|
| **[[glossary\|Anti-Personnel (Apers)]]** | 3 (green) | 3 |
| **[[glossary\|Anti-Tank (Atk)]]** | — | — |
| **Assault** | 3 | 3 |
| **[[force-elements\|Troop Quality (TQ)]]** | 4 | 3 |
| **Resilience** | 3 → 2 | 1 |

| Attribute | Value |
|-----------|-------|
| [[force-elements\|Force Element (FE)]] Type | **F** (green diamond — Foot) |
| NATO symbol | Infantry (☒ crossed rifles) |
| Range (Anti-Personnel (Apers)) | 2,000m |
| Size | [[force-elements\|Platoon (PL)]] (3 dots) |
| Can target | Foot (green), Wheeled (grey) |
| Targeted by | Anti-Personnel (Apers) (green) only |

**vs NATO Foot [[force-elements|Platoon (PL)]]:**
- [[force-elements|Troop Quality (TQ)]] 4 vs NATO 6 — significantly worse firing and Rally [[glossary|Dice Roll Modifiers (DRMs)]]
- Resilience 3 vs NATO 4 — Eliminated after 3 Hits instead of 4
- Same Firepower and Assault values

**Key traits:**
- Same terrain advantages as NATO Foot (Dense Wood access, Cover, Shell Scrapes, Vehicle Vulnerability bonus)
- Lower [[force-elements|Troop Quality (TQ)]] means worse [[glossary|Dice Roll Modifiers (DRMs)]] on everything — fires, assaults, Rally tests
- Lower Resilience means one fewer Hit before Elimination
- Shares an ID with a Tracked [[force-elements|Platoon (PL)]] (same mechanised infantry platoon)

**Scenario 1 examples:** 1-A-1 F, 2-A-1 F, 1-B-1 F, 2-B-1 F, 3-A-1 F, 3-B-1 F (all have RPG ATGMs)

### Main Battle Tank Platoon ([[force-elements|MBT]])

Russian [[force-elements|Main Battle Tank (MBT)]] with reactive armour (gold type — hard to kill). However, its main gun has **purple Anti-Tank (Atk)** (not gold) — it can destroy [[force-elements|Armoured Personnel Carriers (APCs)]]/[[force-elements|Infantry Fighting Vehicles (IFVs)]] but **cannot penetrate NATO [[force-elements|MBT]] armour** with direct fire alone. Needs [[atgm|Stabber Anti-Tank Guided Missiles (ATGMs)]] to engage NATO [[force-elements|MBTs]].

| Stat | Full | Half Resilience |
|------|------|----------------|
| **[[glossary\|Anti-Personnel (Apers)]]** | 2 (green) | 2 |
| **[[glossary\|Anti-Tank (Atk)]]** | 3 (purple) | 3 |
| **Assault** | 1 | 1 |
| **[[force-elements\|Troop Quality (TQ)]]** | 4 | 3 |
| **Resilience** | 3 → 2 | 1 |

| Attribute | Value |
|-----------|-------|
| [[force-elements\|Force Element (FE)]] Type | **T** (gold square — [[force-elements\|Main Battle Tank (MBT)]] armour) |
| NATO symbol | Armour (rectangle with diagonal line) |
| Range (Anti-Tank (Atk)) | 2,000m |
| Size | [[force-elements\|Platoon (PL)]] (3 dots) |
| Can target | Foot, Wheeled, Tracked-purple — but **NOT** gold-type (NATO [[force-elements\|Main Battle Tanks (MBTs)]]) |
| Targeted by | Gold Anti-Tank (Atk) only (or [[atgm\|ATGMs]]) |

> [!warning] Critical asymmetry
> Russian [[force-elements|MBTs]] have **gold armour** (hard to kill — only NATO gold Anti-Tank (Atk) or [[atgm|ATGMs]] can hit them) but only **purple Anti-Tank (Atk)** (cannot penetrate NATO [[force-elements|MBT]] armour). This means:
> - Russian [[force-elements|MBTs]] are **immune** to NATO [[force-elements|IFV]]/[[force-elements|APC]] fire
> - Russian [[force-elements|MBTs]] **cannot** kill NATO [[force-elements|MBTs]] with their main gun
> - Stabber [[atgm|ATGMs]] are **essential** for engaging NATO [[force-elements|MBTs]]
> - NATO [[force-elements|MBTs]] with gold Atk 4 are the primary counter to Russian [[force-elements|MBTs]]

**Key traits:**
- Does not carry Mounted Foot [[force-elements|Platoons (PLs)]] (no [[force-elements|Armoured Personnel Carrier (APC)]] icon) — pure armour
- Standalone platoon (no paired Foot [[force-elements|Platoon (PL)]])
- Anti-Tank (Atk) 3 on the purple column is still powerful against non-[[force-elements|MBT]] targets
- In larger scenario [[glossary|Order of Battle (ORBAT)]]s (not Scenario 1 starter)

**Identified IDs:** 1-A-2 (from rulebook [[glossary|Order of Battle (ORBAT)]] examples)

## Russian [[atgm|Anti-Tank Guided Missiles (ATGMs)]]

| [[atgm\|Anti-Tank Guided Missile (ATGM)]] | Range | Font | Restriction |
|------|-------|------|-------------|
| **RPG** | Short (1,000m) | Red | Any Foot [[force-elements\|Force Element (FE)]] can carry |
| **Stabber** | 3,000m | Red | **Only Russian Tracked [[force-elements\|Force Elements (FEs)]]** |

Both function as **gold Anti-Tank (Atk)** when fired (can target any vehicle including [[force-elements|Main Battle Tanks (MBTs)]]). See [[atgm]].

> [!tip] Stabbers are critical
> Since Russian [[force-elements|MBTs]] have purple (not gold) Anti-Tank (Atk), the Stabber [[atgm|Anti-Tank Guided Missile (ATGM)]] is the **only way** for Russian ground forces to engage NATO [[force-elements|MBTs]] with direct fire. In most [[glossary|Order of Battle (ORBAT)]]s, Stabbers start **Depleted** (one shot only). Protect them and use them wisely.
