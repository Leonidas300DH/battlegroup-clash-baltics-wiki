# Game Modes

Battlegroup Clash: Baltics (BC:B) has three play modes, each building on the previous. You must know Basic mode to play Standard, and Standard to play Advanced. In the rulebook, Standard mode rules are in **red shaded boxes**, Advanced mode in **blue shaded boxes**.

> [!tip]
> Modes can be treated as modules — e.g. Basic mode + Morale rules, or Standard mode + Ammo rules from Advanced.

## Basic Mode

The entry point. Players read the scenario, set up, and jump straight into action.

**What's included:**
- All movement, Direct Fire, Indirect Fire, and Assault mechanics
- [[terrain|Terrain]] effects, [[line-of-sight|Line of Sight]], Cover
- [[support-assets|Support Assets]] (Artillery, [[support-assets|First Person View drone (FPV)]], [[support-assets|Unmanned Aerial Support drone (UAS)]])
- [[electronic-warfare|Electronic Transmission (ETX) generation]] (simplified version)
- All 8 core [[actions-overview|Actions]]

**What's excluded:**
- No [[sync-matrix|Sync Matrix]] or [[orders|Orders]]
- No [[fire-plans|Fire Plans]]
- No [[morale|Morale]] or Rally tests
- No [[command-activation|Command Activations]]
- No Ammo tracking
- No doctrinal mechanics

**Best for:** players new to wargaming, time-limited sessions, learning the core mechanics.

## Standard Mode

The **recommended mode**. Adds realism through planning and morale without excessive complexity.

**Adds to Basic:**
- **[[sync-matrix|Sync Matrix]]** — write Orders and Fire Plans for the entire game before Turn 1
- **[[orders|Orders]]** — Task Groups must follow their assigned Orders; deviation requires [[orders#New Orders|New Orders]]
- **[[fire-plans|Fire Plans]]** — pre-planned target areas reduce ETX costs for Indirect Fire
- **[[morale|Morale]]** — FEs can become Suppressed, Disrupted, or Broken
- **[[morale#Rally Tests|Rally Tests]]** — recover from Suppression/Disruption in the Command Phase
- **[[command-activation|Command Activations]]** — [[force-elements|Headquarters (HQ)]] [[force-elements|Force Elements (FEs)]] can activate additional Task Groups simultaneously
- **Modified [[electronic-warfare|ETX generation]]** — different ETX amounts based on Fire Plans
- **EW Chits 6A/B and 7A/B** added to the draw pool (only 5 chits in Basic)

**Does NOT change:** ETX generation uses different rules from Basic, as described on Player Aid 1.

## Advanced Mode

Maximum realism. Adds doctrinal asymmetry between NATO and Russia, plus logistics.

**Adds to Standard:**
- **[[mission-command|Mission Command]]** (NATO only) — one Task Group per turn can receive New Orders without Reorg penalty or [[electronic-warfare|Electronic Transmission (ETX)]]
- **[[red-doctrine|Red Doctrine]]** (Russia only) — Russian Task Groups/[[support-assets|Support Assets (SAs)]] **cannot deviate** from their pre-game Orders or Fire Plans. No New Orders, no flexibility
- **[[ammo|Ammo]]** — [[force-elements|Platoon (PL)]]-sized [[force-elements|Force Elements (FEs)]] have limited Ammo; Direct Fire costs 1, Assault costs 2; at 0 Ammo, Firepower/Assault Value = 1
- **[[replendem|REPLENDEM]]** action — A1 Echelon [[force-elements|FEs]] can resupply Ammo and [[atgm|Anti-Tank Guided Missiles (ATGMs)]]
- **A1 Echelon [[task-groups|Task Groups]]** added to [[glossary|Order of Battle (ORBAT)]]s
- **Fire Mission / Ammo Card** replaces Fire Mission markers for tracking
- **Russia draws 2 [[electronic-warfare|Electronic Warfare (EW)]] Chits, keeps 1** (compensating for doctrinal rigidity)
- Russia gets additional Fire Missions for artillery and [[support-assets|First Person View drone (FPV)]]

### Advanced Mode Asymmetry

| Aspect | NATO | Russia |
|--------|------|--------|
| Doctrine | [[mission-command\|Mission Command]] — flexible | [[red-doctrine\|Red Doctrine]] — rigid |
| Order changes | Free (1 [[task-groups\|Task Group (TGp)]]/turn via Mission Command) | **Forbidden** |
| Move-Fire penalty | -1 [[glossary\|Dice Roll Modifier (DRM)]] | -2 [[glossary\|DRM]] |
| Resilience (typical [[force-elements\|PL]]) | 4 | 3 |
| [[force-elements\|Troop Quality (TQ)]] (typical) | Higher | Lower |
| Fire Missions | Fewer | More (especially artillery/[[support-assets\|FPV]]) |
| [[electronic-warfare\|EW]] Chit draw | 1 chit | 2 chits, keep 1 |
