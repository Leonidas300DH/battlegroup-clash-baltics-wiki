# Preparation Phase (Phase 1)

The Preparation Phase occurs **once** before the game begins. It covers setup and planning.

## Steps

### 1.1 Read Scenario Instructions and Set-Up

1. Agree on scenario and sides
2. Read your side's Scenario Booklet (keep it secret from the opponent — Fog of War)
3. Note your [[glossary|Order of Battle (ORBAT)]] — [[force-elements|Force Elements (FEs)]], [[support-assets|Support Assets (SAs)]], [[task-groups|Task Group]] assignments
4. **Determine Secondary Objective** — roll secretly on the Commander's Intent table
5. **Set-up the board**:
   - Place map(s)
   - Shuffle [[electronic-warfare|ETX markers]] face-down within reach
   - Place EW Chits in draw pool (remove 6A/B and 7A/B in Basic mode)
   - Set aside Information markers, Target markers, dice
   - Place Turn Tracker Die on '1'
   - Set up FE and SA counters per scenario instructions
   - Optionally [[stacking|Mount/Embed]] FEs during setup
   - Place [[support-assets|Artillery]], [[support-assets|First Person View drone (FPV)]] off-map; set Fire Mission markers
   - [[support-assets|Unmanned Aerial Support drone (UAS)]] begins off-map (placed on first Move Action)

### 1.2 Produce a Sync Matrix (Standard/Advanced)

See [[sync-matrix]] for full details. Each side writes [[orders|Orders]] and [[fire-plans|Fire Plans]] for every turn of the game.

## Stacking During Setup

Before the game begins, sides can:
- **Mount** a Foot [[force-elements|Platoon (PL)]] [[force-elements|Force Element (FE)]] into a Tracked [[force-elements|PL]] [[force-elements|FE]] (with [[force-elements|Armoured Personnel Carrier (APC)]]/[[force-elements|Infantry Fighting Vehicle (IFV)]] icon)
- **Embed** a [[force-elements|Section (SECT)]] [[force-elements|FE]] into a Tracked [[force-elements|PL]] [[force-elements|FE]]

This is done during setup without needing a Move Action. See [[stacking]].
