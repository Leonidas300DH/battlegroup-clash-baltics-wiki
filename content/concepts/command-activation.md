# [[force-elements|Headquarters (HQ)]] Command Activation (Standard/Advanced Mode)

When a [[task-groups|Task Group]] containing an **[[force-elements|HQ]] [[force-elements|Force Element (FE)]]** is activated, the activating side may optionally activate **additional Task Groups** simultaneously.

## How Many?

**X additional Task Groups**, where X = [[force-elements|HQ]]'s **[[glossary|Command & Control (C2)]] Rating minus 1**.

> Example: An [[force-elements|HQ]] with [[glossary|C2]] Rating 3 can activate 2 additional Task Groups.

## Range Restrictions

| HQ Type | Range Limit |
|---------|------------|
| **Main [[force-elements|HQ]]** | **Any** Task Group on the map |
| **Tac [[force-elements|HQ]]** | Only Task Groups with at least **one [[force-elements|FE]] within 1,000m** of the Tac [[force-elements|HQ]] at the time of activation |

> Only one [[force-elements|FE]] in the target Task Group needs to be within 1,000m for the Task Group to qualify.

## Activation Rules

- All Command-Activated Task Groups are considered activated **at the same time**
- The ordering of individual [[force-elements|FE]] activations across all activated Task Groups is at the discretion of the acting side
- [[force-elements|FEs]] from different Task Groups may participate in the **same [[assault|Assault]]** this way
- [[force-elements|FEs]] must still follow their **own Task Group's [[orders|Orders]]** — the Command Activation does not override Order requirements

## Cost

The opponent draws **1 [[electronic-warfare|Electronic Transmission (ETX)]] marker** per additional Task Group Command-Activated.

## Restrictions

- Command Activation can only activate **Task Groups** — not [[support-assets|Support Assets (SAs)]]
- Only available in **Standard and Advanced mode**

## FAQ: HQ Off-Map

> [!note]
> An [[force-elements|HQ]] can use its [[glossary|C2]] Rating and Command Activation even if **not yet on the map**, provided its Sync Matrix shows it is scheduled to enter on the current turn. For a Tac [[force-elements|HQ]] entering the map, only Task Groups with [[force-elements|FEs]] within 1,000m of the **entry point** qualify for Command Activation.

## Tactical Value

Command Activations enable:
- **Coordinated assaults**: multiple Task Groups attacking the same objective simultaneously
- **Synchronized maneuver**: moving multiple elements at once to overwhelm the opponent's ability to Reactive Fire
- **Tempo advantage**: activating 3+ Task Groups in a single "turn" while the opponent only activates one
