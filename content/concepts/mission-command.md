# Mission Command (Advanced Mode — NATO Only)

Mission Command is [[nato|NATO's]] doctrinal advantage in Advanced mode. It allows one [[task-groups|Task Group]] per turn to receive [[orders#New Orders|New Orders]] **without** the usual penalties.

## Normal New Orders Cost

Without Mission Command, issuing New Orders:
- Places a **Reorg marker** on all [[force-elements|Force Elements (FEs)]] in the Task Group (cannot activate that turn)
- Generates **1 [[electronic-warfare|Electronic Transmission (ETX)]]** for the opponent
- Effectively loses an entire turn for that Task Group

## Mission Command Benefit

During the [[command-phase|Command Phase]] New Orders step, NATO may issue New Orders to **one** Task Group with:
- **No Reorg markers** — the Task Group can act immediately
- **No [[electronic-warfare|ETX]]** generated for the opponent

> [!note] Design note
> NATO doctrine seeks to empower decision-making on the ground. Mission Command abstractly represents the flexibility and initiative that NATO commanders at the tactical level exercise — adapting to changing situations without waiting for higher authority.

## Restrictions

- Only **one** Task Group per turn benefits from Mission Command
- Only [[nato|NATO]] has this ability — [[russia|Russia]] is constrained by [[red-doctrine|Red Doctrine]]
- Only available in **Advanced mode**

## Strategic Value

Mission Command is NATO's answer to the chaos of battle. When your plan meets the enemy:
- React to unexpected enemy movements without losing a turn
- Redirect a Task Group to exploit an opportunity
- Recover from a setback without the [[electronic-warfare|ETX]] and Reorg penalty
