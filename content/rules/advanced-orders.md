# Advanced Orders — Battlefield Effects (1.2.1.3)

From the [[ew-adv-supp-rules|Advanced Rules Supplement]]. This rule gives **mechanical game effects** to [[orders|Order]] verbs. In the base game, Orders have no mechanical consequence — this changes that.

> [!important]
> With this rule, certain Actions (Overwatch, Dig Shell Scrape) now **require** specific Orders. A [[task-groups|Task Group (TGp)]] without the right Order cannot take those Actions.

## Order Verbs and Their Effects

| Order Verb | Effect |
|------------|--------|
| **Deny / Disrupt / Fix / Hold / Secure** | [[force-elements|Force Elements (FEs)]] in the [[task-groups|TGp]] **can** be given an [[overwatch|Overwatch]] Action. Without one of these Orders, Overwatch is **not available**. |
| **Deny / Hold** | [[force-elements|FEs]] in the [[task-groups|TGp]] **can** be given a [[dig-shell-scrape|Dig Shell Scrape]] Action. Without one of these Orders, Dig Shell Scrape is **not available**. |
| **Attack / Seize** | [[force-elements|FEs]] in the [[task-groups|TGp]] get **+1 [[glossary|Dice Roll Modifier (DRM)]]** for [[direct-fire|Direct Fire]] and [[assault|Assault]] rolls against enemy [[force-elements|FEs]] occupying an **objective designated in the [[sync-matrix|Sync Matrix]]**. |
| **Breach** | [[force-elements|FEs]] in the [[task-groups|TGp]] **ignore [[dig-shell-scrape|Shell Scrape]] marker effects** when Assaulting an enemy [[force-elements|FE]] blocking the way to a designated objective. **Only one [[task-groups|TGp]] per side** may have Breach (can apply across multiple Turns). |
| **Exploit** | If on the **previous Turn**, [[force-elements|FEs]] from this [[task-groups|TGp]] (or a friendly [[task-groups|TGp]] within 1,000m) achieved a green [[assault|Assault]] result: the [[task-groups|TGp]] receives free [[orders#New Orders|New Orders]] via [[mission-command|Mission Command]] rules. For [[nato|NATO]], this is **in addition** to the normal once-per-turn Mission Command. For [[russia|Russia]], this **overrides [[red-doctrine|Red Doctrine]]** for this [[task-groups|TGp]]. Also: any [[force-elements|FE]] given a Move or [[move-fire|Move-Fire]] Action gets **+50% Movement Allowance**. If no qualifying Assault occurred, the [[task-groups|TGp]] uses the previous Turn's Order or needs New Orders. |
| **Raid** | [[force-elements|FEs]] in the [[task-groups|TGp]] get **doubled Movement Allowance** across any Terrain (except No-Go) when taking a [[move-fire|Move-Fire]] Action against a target/area designated in the [[sync-matrix|Sync Matrix]]. |
| **Screen** | [[force-elements|ISTAR]] [[force-elements|FEs]] in the [[task-groups|TGp]] give **+1 [[glossary|DRM]]** on [[indirect-fire|Indirect Fire]] if they have [[line-of-sight|Line of Sight (LoS)]] to the target. [[direct-fire|Direct Fire]] against any [[force-elements|FE]] in the [[task-groups|TGp]] receives **-1 [[glossary|DRM]]** (harder to hit). These [[glossary|DRMs]] are **additional** to those on Player Aids 3 and 4. |
| **Withdraw** | [[force-elements|FEs]] in the [[task-groups|TGp]] get **doubled Movement Allowance** across any Terrain (except No-Go) when moving **away** from enemy [[force-elements|FEs]] in [[line-of-sight|LoS]]. |

## Key Implications

### Overwatch is now restricted
In the base game, any [[force-elements|FE]] can take [[overwatch|Overwatch]]. With this rule, only [[task-groups|TGps]] with **Deny, Disrupt, Fix, Hold, or Secure** Orders can use Overwatch. An attacking [[task-groups|TGp]] (with Seize or Attack) **cannot set up Overwatch**.

### Dig Shell Scrape is now restricted
Only [[task-groups|TGps]] with **Deny or Hold** Orders can dig in. An advancing force cannot create new defensive positions.

### Exploit enables Russian flexibility
The **Exploit** Order is the only way Russian forces can get free [[orders#New Orders|New Orders]] under [[red-doctrine|Red Doctrine]] — but only after a successful Assault. This makes Assaults even more tactically significant.

### Breach counters entrenched defenders
A **Breach** [[task-groups|TGp]] ignores [[dig-shell-scrape|Shell Scrape]] DRMs (-1/-2) when Assaulting, making it the answer to heavily fortified positions.

> [!tip]
> Players using custom Order Verbs can agree in advance what their mechanical effects are.
