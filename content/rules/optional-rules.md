# Optional Rules

BC:B includes four optional rules that can be used with any scenario and any [[game-modes|game mode]].

## Weather

Roll 1D6 in the Preparation Phase (before drafting the [[sync-matrix|Sync Matrix]]) or agree on weather:

| Roll | Weather | Effects |
|------|---------|---------|
| 1-4 | **Fair** | No effect |
| 5 | **Low clouds, high winds** | [[support-assets|First Person View drone (FPV)]] Fire Missions halved (round down); [[support-assets|Unmanned Aerial Support drone (UAS)]] covers only 1 Grid Square (placed in center); [[support-assets|UAS]] can only move 2 Grid Squares |
| 6 | **Storms** | No [[support-assets|UAS]] or [[support-assets|FPV]] used; remove [[electronic-warfare|Electronic Warfare (EW)]] Chits 1A/B and 4A/B; [[line-of-sight|Line of Sight (LoS)]] reduced to 2,000m (3,000m for Recce); off-road Wheeled movement reduced by 50% |

## Flexible Task Groups

### Preparation Phase — Creating Task Groups
Each side can create **custom Task Groups** from the [[glossary|Order of Battle (ORBAT)]]'s [[force-elements|Force Elements (FEs)]] and [[support-assets|Support Assets (SAs)]], instead of using the pre-set groupings. Done after Commander's Intent is determined. Requires a separate sheet noting assignments.

### Command Phase — Reassigning Between Task Groups (Standard/Advanced)
As a New Order, an individual [[force-elements|FE]] can be **Reassigned** to a different Task Group:
- Only the [[force-elements|FE]] being Reassigned receives a Reorg marker (not the whole Task Group)
- Both the source and destination Task Groups may activate normally
- On the subsequent turn, the [[force-elements|FE]] follows the new Task Group's Orders
- Opponent draws **1 [[electronic-warfare|Electronic Transmission (ETX)]]** per [[force-elements|FE]] Reassigned
- [[red-doctrine|Red Doctrine]]: Russia **cannot** Reassign during the Command Phase

## More Smoke

After placing a [[indirect-fire#Smoke|Smoke marker]] (step 2), roll 1D6 to place a **second Smoke marker** touching the first:

| Roll | Placement |
|------|-----------|
| 1 | Directly above (touching) |
| 2 | Directly right (touching) |
| 3 | Directly below (touching) |
| 4 | Directly left (touching) |
| 5-6 | Firing player's choice (must touch, can be on corner) |

Both markers count as a **single Partial Obstruction** and are removed in Clean-up.

## LoS Through Partial Obstacles

In the standard rules, [[line-of-sight|Line of Sight (LoS)]] can be drawn through a single [[line-of-sight#Partial Obstructions|Partial Obstruction]] without limit. This optional rule adds:

**LoS cannot be drawn more than 250m through a single Partial Obstruction.**

Use if large single-piece Partial Obstructions feel unrealistically transparent.
