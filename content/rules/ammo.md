# Ammo (Advanced Mode)

In Advanced mode, **[[force-elements|Platoon (PL)]]-sized [[force-elements|Force Elements (FEs)]]** have a limited Ammo supply that is consumed by combat actions and tracked on the Fire Mission / Ammo Card.

## Ammo Consumption

| Action | Ammo Cost |
|--------|-----------|
| [[direct-fire\|Direct Fire]] | 1 (not if using an [[atgm\|Anti-Tank Guided Missile (ATGM)]]) |
| [[assault\|Assault]] (Engaging) | 2 |

> [!note]
> [[force-elements|Sections (SECTs)]] also use Ammo in reality, but the amounts are inconsequential for game purposes and are not tracked.

## Effects of Low/No Ammo

| Ammo Remaining | Direct Fire Effect | Assault Effect |
|----------------|-------------------|----------------|
| 1+ | Normal | Normal |
| 0 | Uses **'1' column** on Results Table (regardless of Firepower Value) | Assault Value = **1** |
| < 2 at Assault start | N/A | Assault Value = **1** |

## Ammo Limits

| [[force-elements|FE]] Type | Maximum Ammo |
|---------|-------------|
| Foot [[force-elements|PL]] | 4 |
| Tracked [[force-elements|PL]] (Vehicle) | 5 |

## Tracking

Ammo is tracked using the **Fire Mission / Ammo Card**:
- Each PL-sized FE has a dedicated **Ammo Tracker** marker
- Placed on the Card in the box corresponding to its Task Group
- Starting Ammo is set by the scenario ORBAT
- Each time Ammo is used, the marker moves leftward toward '0'

> [!note]
> The Card replaces Fire Mission markers in Advanced mode, preventing map clutter.

## Resupply

Ammo can be resupplied via the [[replendem|REPLENDEM]] Action by A1 Echelon [[force-elements|FEs]]. [[atgm|ATGMs]] can also be resupplied this way.

## No Ammo Type Distinction

BC:B does **not** distinguish between Ammo types — whether firing Apers or Atk, it costs 1 Ammo either way.

## Strategic Implications

Ammo scarcity forces players to:
- **Prioritize targets** — don't waste shots on low-probability attacks
- **Plan resupply routes** — A1 Echelons need to reach forward units safely
- **Protect A1 Echelons** — if Eliminated, all carried supplies are lost
- **Time Assaults carefully** — 2 Ammo per Assault is expensive
