## FAQ Answers

### [[atgm|ATGMs]]

| Question | Answer |
|----------|--------|
| Can an [[force-elements|FE]] fire an [[atgm|ATGM]] and standard Direct Fire simultaneously? | **No.** Firing the [[atgm|ATGM]] counts as the [[force-elements|FE]]'s entire Direct Fire action for the turn. |
| Can an A1 Echelon fire an [[atgm|ATGM]] it carries? | **No.** A1 Echelons represent transport, not combat units. |

### [[electronic-warfare|Electronic Warfare (EW)]] Activity

| Question | Answer |
|----------|--------|
| Is step 3 (draw [[electronic-warfare|EW]] Chit) only on Turn 1, or every turn? | **Every turn.** On Turn 1 it is the only step applied (skip steps 1, 2, 4, 5). |

### [[force-elements|Headquarters (HQs)]]

| Question | Answer |
|----------|--------|
| Can an [[force-elements|HQ]] use [[glossary|Command & Control (C2)]]/Command Activation if off-map? | **Yes**, if its Sync Matrix schedules it to enter on the current turn. Tac [[force-elements|HQ]]: only [[task-groups|Task Groups (TGps)]] with [[force-elements|FEs]] within 1,000m of the entry point qualify. |

### [[line-of-sight|Line of Sight]]

| Question | Answer |
|----------|--------|
| Can [[force-elements|FEs]] on hills ignore Obstruction/Partial Obstruction? | **No.** Hills only block [[line-of-sight|LoS]] that crosses them. Being elevated provides no additional [[line-of-sight|LoS]] advantage. |

### Assaults

| Question | Answer |
|----------|--------|
| Do Assaults follow normal Movement rules? | **No.** [[force-elements|FEs]] are "placed" at the Assault Location. Movement restrictions don't apply in the same way. |
| Does Reactive Fire during Assault require [[line-of-sight|LoS]]? | **Yes.** Normal Reactive Fire rules apply (Overwatch marker + [[line-of-sight|LoS]] required). |
| What if a [[force-elements|FE]] becomes Disrupted via Reactive Fire during an Assault? | If it's the **only attacking Foot [[force-elements|FE]]**, Assault ends and Attackers Pull Back. Otherwise, the Disrupted [[force-elements|FE]] participates with reduced [[force-elements|Troop Quality (TQ)]]. |
| Can [[force-elements|FEs]] with Reorg markers Pull Back from an Assault? | **Yes.** Pull Back is not an Activation. If they Engage instead, Attacker gets +2 [[glossary|Dice Roll Modifier (DRM)]] (disorganized defenders). |
| Do Assault Hits cause Suppression? | **No.** Hits from the Assault result have no Morale effect. But all [[force-elements|FEs]] on the losing side that Pull Back after Engaging receive a Disrupted marker. |
| Can Dismounted Foot [[force-elements|FEs]] Mount at end of Assault? | **No.** Mounting only occurs at the end of Move or Move-Fire Actions. Participating [[force-elements|FEs]] can Reposition up to 250m. |
