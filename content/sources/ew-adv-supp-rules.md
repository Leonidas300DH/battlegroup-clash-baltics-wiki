# Advanced Rules Supplement: Orders & Electronic Warfare

**File:** `raw/EWAdvSupp_Rules.pdf`
**Pages:** 6
**Designer:** James Buckley
**Sponsor:** Arondite

## Key Takeaways

This supplement adds three **modular** rule sets to Advanced mode. Players can pick and choose which to use:

1. **[[advanced-orders|Battlefield Effects of Orders]]** — Order verbs now have mechanical game effects (DRM bonuses, movement modifiers, action restrictions)
2. **[[boundary-lines|Boundary Lines]]** — [[task-groups|Task Groups (TGps)]] must operate within assigned geographic areas; crossing generates [[electronic-warfare|Electronic Transmission (ETX)]]
3. **[[advanced-ew|Advanced Electronic Warfare]]** — completely **replaces** the base game's [[electronic-warfare|EW Chit]] system with Electronic Warfare Points (EWP) assigned across 4 EW types: SIGINT, GNSS Jamming, Drone Jamming, and Comms Jamming

## What Changes vs Base Game

| Aspect | Base Game | This Supplement |
|--------|-----------|-----------------|
| Order effects | No mechanical effect | 10 Order verbs with specific DRM/action effects |
| Geographic control | None | [[boundary-lines\|Boundary Lines]] restrict [[task-groups\|TGp]] movement areas |
| [[electronic-warfare\|EW]] system | Random EW Chits + Target markers as fire DRM | Strategic EWP allocation across 4 EW Types |
| Target markers | From ETX, +DRM on any fire | From SIGINT only, on identified [[force-elements\|Headquarters (HQ)]]/[[force-elements\|ISTAR]] [[force-elements\|Force Elements (FEs)]], Indirect Fire only |
| EW planning | None (random draw) | Full EW Plan on [[sync-matrix\|Sync Matrix]], decided in Preparation Phase |
| Self-jamming | Not possible | Drone and Comms Jamming can affect the attacker |

## New Components Required

- **Player Aid 7** — Order Verb effects, Boundary Lines, EW Activity sequence
- **Player Aid 8** — EW Attack procedures for all 4 types, Target marker rules
- **Advanced Sync Matrix** — larger, with EW Plan boxes (A/D for each EW Type per Turn)
